# Problems and instances

Vamos pegar um exemplo de multiplicação de 981 by 1234. Os algoritmos não só poveem uma forma de gerar a multiplicação para esses dois particulares níumeros. Em fato, eles dão a solução geral para o problema de multiplicar dois números positivos

- Nós dizemos que (981, 1234) é uma instância desse problema

Multiplicar 789 por 9742, que nós podemos também expressar por (789, 9742) é outra instância do problema de multiplicação

<note>
Os problemas mais interessantes possuem coleções infinitas de instâncias
